# casino
Early project for experimenting with OOP and sorting algorithms. A cli-based game, written in Java. Sorting algorithms include: insertion sort, selection sort, merge sort, and quick sort.

```
Option [help]:
Willkommen im Spielcasino "Las Vegas"!
Ihnen stehen folgende Befehle zur Verfügung:

    start   -   Startet ein neues Spiel
    show    -   Deckt die eigenen Karten auf
    shuffle -   Mischt die eigenen Karten
    sort    -   Sortiert die eigenen Karten
    ende    -   Beendet das Spiel
    help    -   Zeigt diese Befehlsübersicht an

Nutzen Sie nun die Eingabeaufforderung um einen weiteren Befehl auszuführen:
[]

Option [start]:
Sie starten ein neues Spiel. Der Dealer fängt an Tisch 23 fängt an die Spielkarten zu mischen. Nach einem wilden Auftritt, bei dem die Spielkarten blind hinter dem Rücken des Dealers gemischt wurden, während dieser barfuß über heisse Kohlen lief, werden nun jedem Spieler fünf Karten ausgeteilt.
Sie haben Ihre Karten erhalten. Das Spiel kann nun beginnen.

Nutzen Sie nun die Eingabeaufforderung um einen weiteren Befehl auszuführen:
[]

Option [show]:
Natürlich wollen Sie wissen, welche Spielkarten Ihnen  ausgeteilt wurden.
Neugierig werfen Sie einen Blick auf Ihre Karten:
    3 7 1 B K A

Nutzen Sie nun die Eingabeaufforderung um einen weiteren Befehl auszuführen:
[]

Option [shuffle]:
Die Reihenfolge Ihrer Spielkarten gefällt Ihnen nicht. Sie fangen an Ihre Karten neu zu mischen.

Nutzen Sie nun die Eingabeaufforderung um einen weiteren Befehl auszuführen:
[]

Option [sort]:
Sie haben eine Pechsträhne und werden langsam unglücklich. Doch nach einem genauen Hinsehen fällt Ihnen auf, dass Sie viel erfolgreicher sein könnten, wenn doch nur Ihre Karten besser sortiert wären.

    Es gibt zwei Arten von Sortierverfahren:

        1. Alle Karten werden auf einmal sortiert.
        2. Jede Karten wird nacheinander sortiert.

    Treffen Sie nun Ihre Auswahl:
    []

    Es gibt zudem drei Sortieralgorithmen:

        1. Sortieren durch Mischen
        2. Sortieren durch Austauschen
        3. Sortieren durch Einfügen

    Wählen Sie nun Ihren bevorzugten Sortieralgorithmus:
    []

Ihre Spielkarten sind nun sortiert und mit neuer Hoffnung sehen Sie dem verbleibenden Spielverlauf entgegen.

Nutzen Sie nun die Eingabeaufforderung um einen weiteren Befehl auszuführen:
[]

Option [end]:
Sie haben genug gespielt für heute und werden langsam müde. Zeit das Spiel zu beenden.
```
